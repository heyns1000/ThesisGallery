# PayPal Integration System

## Overview

This is an enterprise-scale PayPal integration system designed to manage payment processing for 7,000+ products across multiple business sectors. The application provides a centralized admin dashboard for managing PayPal products, subscription plans, and bulk operations. It features a Flask-based backend with PostgreSQL database, Redis-powered background job processing via Celery, and comprehensive webhook handling for real-time PayPal event processing.

## User Preferences

Preferred communication style: Simple, everyday language.

## System Architecture

### Backend Architecture
- **Framework**: Flask web application with SQLAlchemy ORM
- **Database**: PostgreSQL with connection pooling and automatic reconnection
- **Background Processing**: Celery task queue with Redis as message broker
- **Session Management**: Flask sessions with configurable secret keys
- **Proxy Support**: ProxyFix middleware for deployment behind reverse proxies

### Data Model Design
- **Hierarchical Structure**: Sectors → Brands → SubNodes for organizing 7,000+ products
- **PayPal Integration Models**: Separate entities for PayPal products, plans, and subscriptions
- **Bulk Operations**: Job tracking system for managing large-scale operations
- **Audit Trail**: Comprehensive logging for all database changes and API interactions
- **Webhook Events**: Dedicated model for handling PayPal webhook notifications

### Background Job System
- **Batch Processing**: Configurable batch sizes for PayPal API rate limit compliance
- **Retry Logic**: Automatic retry with exponential backoff for failed operations
- **Progress Tracking**: Real-time job status monitoring with detailed error reporting
- **Rate Limiting**: Built-in delays between API requests to respect PayPal limits

### Frontend Architecture
- **Template Engine**: Jinja2 templates with Bootstrap 5 dark theme
- **Interactive Dashboard**: Real-time status monitoring with progress bars and metrics
- **Responsive Design**: Mobile-friendly admin interface with Font Awesome icons
- **AJAX Operations**: Asynchronous operations for bulk actions and status updates

### Configuration Management
- **Environment-based**: All sensitive data configured via environment variables
- **Multi-environment Support**: Separate configurations for sandbox and production
- **Database Connection**: Advanced PostgreSQL settings with connection pooling
- **PayPal Settings**: Configurable API endpoints, batch sizes, and retry parameters

## External Dependencies

### Payment Processing
- **PayPal API**: Complete integration with PayPal REST API for products, plans, and subscriptions
- **PayPal SDK**: Server-side SDK for streamlined API interactions
- **Webhook Verification**: PayPal webhook signature validation for security

### Infrastructure Services
- **PostgreSQL**: Primary database for persistent data storage
- **Redis**: Message broker and result backend for Celery task queue
- **Celery**: Distributed task queue for background job processing

### Frontend Libraries
- **Bootstrap 5**: UI framework with dark theme support
- **Font Awesome**: Icon library for enhanced user interface
- **Chart.js**: Data visualization for dashboard metrics and statistics

### Development Tools
- **SQLAlchemy**: ORM with support for complex queries and relationships
- **Flask-SQLAlchemy**: Flask integration with automatic session management
- **Werkzeug**: WSGI utilities including proxy fix middleware
- **UUID**: Unique identifier generation for database entities
